¿Qué es HTML5?

Esto fue posteado el 24/07/2025 por UMG-Guastatoya

Durante los últimos años, el estándar HTML5 se ha estado afianzando en muchas páginas web. Sin embargo, ¿qué queremos decir con esto? ¿Cuál es la diferencia entre HTML "común", por así decirlo, y este nuevo estándar? Hoy hablaremos sobre esto y responderemos a la pregunta de qué es HTML5.

HTML5 es un lenguaje más semántico. En otras palabras, se dota de un significado a las diferentes partes de la página web, de forma que el navegador puede distinguir la parte que forma parte de la cabecera gracias a que hay una etiqueta que dice "Cabecera", bueno, lo dice en inglés: <header>. ¿Para qué sirve? Ayuda a los buscadores como Google a entender tu página web, algo que se espera sea positivo de cara al SEO, y también sirve para que los navegadores muestren de forma inteligente diferentes versiones de la página web sin tener que usar plug-ins para dispositivos específicos.

¿Introducción a CSS3?

Esto fue posteado el 24/07/2025 por UMG-Guastatoya

El nombre hojas de estilo en cascada viene del inglés Cascading Style Sheets, del que toma sus siglas. CSS es un lenguaje usado para definir la presentación de un documento estructurado escrito en HTML o XML (y por extensión en XHTML). El W3C (World Wide Web Consortium) es el organismo encargado de formular la especificación de las hojas de estilo que servirán de estándar para los agentes de usuario o navegadores.

¿Para qué sirve el CSS?

El CSS sirve para definir la estética de un sitio web en un documento externo, y eso mismo permite que modificando ese documento (la hoja CSS) podamos cambiar la estética entera de un sitio web. El mismo sitio web puede variar totalmente de estética cambiando solo la CSS, sin tocar para nada los documentos HTML o JSP o ASP que lo componen.

¿Qué es y para qué sirve JavaScript?

JavaScript (JS) es un lenguaje ligero o interpretado, orientado a objetos con funciones de primera clase, más conocido como el lenguaje de script para páginas web, pero también usado en muchos entornos sin navegador, tales como Node.js.
